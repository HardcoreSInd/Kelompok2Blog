Sejarah Windows 10

Awal Berdirinya

Windows 10 resmi dirilis pada 29 Juli 2015 oleh Microsoft. Sistem operasi ini dikembangkan sebagai penerus Windows 8.1 dan dirancang sebagai platform universal yang bekerja pada berbagai perangkat seperti komputer, laptop, tablet, bahkan Xbox.

Proyek Windows 10 awalnya dikenal sebagai "Threshold" dalam tahap pengembangan sebelum dirilis ke publik.

Diciptakan Oleh Siapa?

Microsoft Corporation

Windows 10 diciptakan oleh Microsoft dengan pimpinan pengembangan oleh Terry Myerson yang saat itu menjabat sebagai kepala divisi sistem operasi Microsoft.

Pendiri Microsoft sendiri adalah Bill Gates dan Paul Allen.

Fungsi dan Kegunaan

Kegunaan Windows 10

  • Sebagai sistem operasi untuk menjalankan aplikasi dan perangkat keras.
  • Mendukung gaming modern dengan fitur DirectX 12.
  • Menghubungkan pengguna dengan layanan cloud Microsoft seperti OneDrive.
  • Digunakan untuk pekerjaan, sekolah, desain, coding, dan multimedia.

Kelebihan & Kekurangan

Kelebihan Windows 10

  • Tampilan modern dan mudah digunakan.
  • Kompatibel dengan banyak software dan hardware.
  • Pembaruan keamanan rutin.
  • Mendukung teknologi gaming modern.

Kekurangan Windows 10

  • Sering melakukan update otomatis yang memakan waktu.
  • Memerlukan spesifikasi perangkat yang cukup tinggi untuk performa optimal.
  • Privasi data sering dipertanyakan karena sistem telemetry Microsoft.
