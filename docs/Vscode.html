Sejarah VS Code

Awal Rilis

Visual Studio Code pertama kali diumumkan pada 29 April 2015 oleh Microsoft. Software ini dikembangkan sebagai editor source code ringan, fleksibel, dan dapat berjalan di berbagai sistem operasi seperti Windows, macOS, dan Linux.

Pada tahun 2016, VS Code dirilis sebagai perangkat lunak open-source menggunakan lisensi MIT License.

Pengembang

Microsoft Corporation

VS Code dikembangkan oleh Microsoft dengan dukungan komunitas internasional melalui platform open-source GitHub. Editor ini berbasis teknologi Electron, yang memungkinkan tampilan modern dan fitur kaya plugin.

Fungsi & Kegunaan

Visual Studio Code digunakan sebagai editor untuk menulis dan menjalankan kode pemrograman. Ini mendukung banyak bahasa pemrograman seperti JavaScript, Python, C++, HTML, Lua, Java, dan lainnya berkat sistem ekstensi plugin. Fitur seperti IntelliSense, Git integration, debugging, dan Live Server menjadikannya pilihan utama programmer modern.

Kelebihan & Kekurangan

Kelebihan

Gratis, open-source, penuh fitur, mendukung ekstensi, ringan dibanding IDE lain, memiliki fitur Auto-complete, Debugging, dan integrasi Git yang sangat baik.

Kekurangan

Menggunakan Electron sehingga dapat memakan RAM cukup besar, beberapa fitur membutuhkan konfigurasi tambahan, dan beberapa bahasa perlu plugin tambahan agar optimal.
