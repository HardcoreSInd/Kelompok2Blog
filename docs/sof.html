Sistem Operasi Aplikasi Desain Software Pemrograman

Sistem Operasi

Windows 10

Jenis: Sistem Operasi

  • Kelebihan: Mudah digunakan, kompatibel luas, banyak dukungan aplikasi dan game.
  • Kekurangan: Berbayar, rawan virus, dan memerlukan spesifikasi tinggi.

Aplikasi Desain & Animasi

Blender

Jenis: Aplikasi Desain & Animasi 3D

  • Kelebihan: Gratis, open source, fitur lengkap (modeling, animasi, rendering, editing).
  • Kekurangan: Antarmuka kompleks, berat di PC spesifikasi rendah.

Software Pemrograman

Visual Studio Code

Jenis: Software Pemrograman

  • Kelebihan: Ringan, banyak ekstensi, mendukung banyak bahasa (JavaScript, Python, dsb).
  • Kekurangan: Butuh ekstensi tambahan untuk fitur lengkap, bisa lambat jika banyak plugin aktif.
